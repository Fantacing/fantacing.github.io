xcode手动下载模拟器

  • 打开console，点击清除

  • 点击模拟器的下载按钮，然后点击取消

  • 在console中找到刚才的操作，找到对应dmg文件的网址，将dmg下载下载

  • 将下载下来的dmg拷贝到~/Library/Caches/com.apple.dt.Xcode

  • 在模拟器下载界面点下载 会自动跳到安装
